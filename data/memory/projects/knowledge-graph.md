# Knowledge Graph

**Status:** Active
**Stack:** Firestore + MCP server

## What it is
Institutional knowledge system for Transparent Partners. Stores structured knowledge
(people, clients, projects, terms) in Firestore and exposes it via a Model Context
Protocol (MCP) server so Claude and other AI tools can query it in real time.

## Goals
- Single source of truth for TP institutional knowledge
- AI-accessible via MCP so Claude can reference it mid-conversation
- Replaces fragmented Notion/Slack/email knowledge silos
